I. INTRODUCTION:

A point-of-sale system is a place where a customer executes the payment for goods or services and where sales taxes may become payable. It is one of if not the cornerstone of any establishment offering various services or products to a variety of customers. A point-of-sale system also plays a vital role for inventory tracking, ensuring that an establishment is able to provide goods or services necessary to its customers.

II. PROBLEM:

Handwritten ordering systems take time and energy and can be proven slow and inefficient in certain establishments. Additionally, handwritten records are easily susceptible to wear and tear over time, making keeping inventory monitoring harder than it should be.

III. OBJECTIVES:

With the presented problem, the group aims to achieve the following:
- Create a simple POS application using C++ that allows users to process sales, select products to process, and provides instant feedback through automatic billing
- The application must also include a simple inventory system to allow users to keep track of item inventory to easily determine the availability of certain products to be purchased
- The application must also allow processing of special discounts (PWD and Senior Discounts) which should be reflected in the application.

double calculateTotal(Order orders[], int orderCount, MenuItem menu[], int menuSize) { double total = 0;
for (int i = 0; i < orderCount; i++) { int id = orders[i].itemID;
int qty = orders[i].quantity;
for (int j = 0; j < menuSize; j++) { if (menu[j].id == id) {
total += menu[j].price * qty; break;
}
}
}
return total;
}

double applyDiscount(double total, bool isSenior, bool isPWD) { double discount = 0.0;
if (isSenior || isPWD) discount = total * 0.20; return discount;
}
